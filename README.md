this is a data analytics project i created using Django rest framework for the backend and react for the frontend it analyses telecommunication data from the 3 top phone service providers in Kenya i.e Safaricom, Airtel and Telcom, it analyses churn rate , complaint insights , customer satisfaction for the various services provided e.g mobile money transfer, data bundle services, voice call services et cetera, it analyzes these based on the regions in Kenya "previous provinces"
